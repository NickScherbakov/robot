Typical Workflow

  1. User sends request via Telegram bot or REST API
  2. Robot validates user access and creates task record
  3. AI processes the request using OpenAI or Mistral API
  4. Cost is calculated and recorded as expense
  5. Response sent back to user
  6. Payment charged if applicable (subscription or micro-payment)
  7. Transaction logged in database for reporting
  8. Daily report generated automatically with profit/loss analysis
